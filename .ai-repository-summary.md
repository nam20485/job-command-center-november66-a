# ProfileGenie Repository Summary

## Project Overview
**ProfileGenie** (Job Command Center) is a LinkedIn Profile Scoring System that automates profile data collection, analysis, and scoring using Playwright for browser automation.

## Tech Stack
- **Language:** C# 12
- **Framework:** .NET 9
- **UI Framework:** Blazor Server
- **Automation:** Microsoft.Playwright (via CDP)
- **Database:** PostgreSQL (via Entity Framework Core)
- **Orchestration:** Designed for .NET Aspire (infrastructure prepared)

## Setup Instructions

### Prerequisites
- .NET 9 SDK
- PostgreSQL database
- Chrome/Chromium browser (for Playwright)

### Build
```bash
cd src
dotnet restore
dotnet build
```

### Run
```bash
# Run the Web application
cd src/ProfileGenie.Web
dotnet run

# Run the Harvester worker
cd src/ProfileGenie.Harvester
dotnet run
```

### Database Setup
```bash
# Create migrations (first time)
cd src/ProfileGenie.Data
dotnet ef migrations add InitialCreate

# Apply migrations
dotnet ef database update
```
